# PID_demo

Demonstration of a PID controller. Tested on Arduino Mega.

![setup](img/setup.jpg)

## Libraries

+ Button: https://github.com/madleech/Button

+ LiquidCrystal_I2C: https://github.com/johnrickman/LiquidCrystal_I2C

## Parts

+ Arduino Mega

+ Sharp 2y0a21 (Infrared Proximity Sensor) 

+ SG90 (Servo Motor)
